# 舞鶴高専 創造技術研究会 (S-ken)

## 🛠️ 私たちについて
私たち「舞鶴高専 創造技術研究会 (S-ken)」は、高専ロボコン優勝を目指し、日々ロボットの設計・製作・制御に取り組んでいます。

メンバー全員で協力し合い、創造性あふれるロボットを開発しています！
